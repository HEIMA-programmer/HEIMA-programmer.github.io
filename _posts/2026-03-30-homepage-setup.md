Finally set up a personal academic homepage using Jekyll + GitHub Pages. The goal is to have a clean, low-maintenance site where I can also keep a research log.

## Why bother?

1. Having a public-facing page helps with visibility in the research community
2. Writing daily logs forces me to reflect on what I actually accomplished
3. It's free and version-controlled with Git

## Tech stack

- **Jekyll** — static site generator, GitHub Pages supports it natively
- **Markdown** — write posts without touching HTML
- **GitHub Pages** — free hosting, auto-builds on push
